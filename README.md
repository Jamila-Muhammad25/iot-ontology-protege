# IoT Ontology using Protégé

## Overview

This repository documents the design and development of an Internet of Things (IoT) ontology using Protégé.

The objective is to model IoT devices, sensors, environments, users, and relationships to support semantic interoperability, reasoning, and intelligent decision-making.

---

## Research Goals

- Build a structured IoT ontology
- Improve interoperability among devices
- Enable semantic reasoning
- Support future smart environment applications

---

## Tools Used

- Protégé
- OWL
- RDF
- SPARQL
- GitHub

---

## Repository Structure

- `ontology/` → ontology files (.owl)
